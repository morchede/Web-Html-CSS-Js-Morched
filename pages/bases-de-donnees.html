Bases de Données

Découvrez les systèmes de gestion de bases de données populaires que j'utilise

MySQL

MySQL est un système de gestion de base de données relationnelle open-source, populaire pour les applications Web.

Oracle

Oracle est un système de gestion de base de données relationnelle largement utilisé dans les entreprises pour la gestion de données complexes.

MongoDB

MongoDB est une base de données NoSQL orientée document, idéale pour les applications flexibles et évolutives.

Firebase

Firebase est une plateforme de développement mobile et web qui fournit des services de bases de données en temps réel et des fonctionnalités back-end.

phpMyAdmin

phpMyAdmin est un outil open-source permettant de gérer MySQL via une interface web, facilitant l'administration de bases de données.
