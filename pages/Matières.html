Projets et TP réalisés

  • Site de Vente de Produits Informatiques - Développement d'un site web pour la vente de produits informatiques avec une interface utilisateur simple, en utilisant HTML, CSS, et JavaScript. Le site permet aux utilisateurs de consulter des produits, de les ajouter au panier et de finaliser leurs achats.
  • Application PHP de gestion d'email - Développement d'une application PHP permettant d'envoyer des emails à partir d'un formulaire. L'application utilise PHP avec la bibliothèque PHPMailer pour envoyer des messages aux destinataires. L'interface inclut des champs pour l'adresse email, le sujet, le message, et permet également l'ajout de fichiers en pièce jointe.
  • Projet Django pour un système de réservation - Création d'une application Django permettant aux utilisateurs de réserver des services en ligne avec gestion de comptes et paiement intégré.
  • Application Web de gestion des réservations avec Angular et Node.js - Utilisation de Angular pour la partie front-end et Node.js pour la gestion des données et des utilisateurs dans une application de réservation en ligne.
  • Application Mobile Native - Météo et Recherche d'Images - Développement d'une application mobile native permettant aux utilisateurs de consulter les prévisions météo en temps réel et de rechercher des images spécifiques de chaque pays. L'application utilise des API pour récupérer les informations météorologiques et les images liées aux pays, et est conçue pour Android ou iOS.
  • Application Flutter pour les informations personnelles - Développement d'une application mobile avec Flutter pour gérer les informations personnelles, utilisant une base de données locale pour la gestion des données.
  • Application de gestion de stock avec Angular et Spring Boot - Création d'une plateforme web avec Angular pour le front-end et Spring Boot pour le back-end, permettant de gérer un stock de produits.
  • Site e-commerce avec Angular et Node.js - Développement d'une application de commerce en ligne avec un front-end Angular et un back-end Node.js pour la gestion des produits, utilisateurs, et commandes.
