Langages de Programmation

Découvrez les langages de programmation que j'utilise

HTML

HTML est le langage standard utilisé pour créer des pages Web. Il définit la structure des documents et leur contenu.

CSS

CSS est utilisé pour styliser les pages Web et contrôler leur mise en page, leurs couleurs et leurs animations.

JavaScript

JavaScript est un langage de programmation permettant d'ajouter des fonctionnalités dynamiques et interactives aux pages Web.

Python

Python est un langage polyvalent, utilisé pour le développement Web, l'analyse de données et l'intelligence artificielle.

Java

Java est un langage de programmation largement utilisé pour le développement d'applications d'entreprise et mobiles (Android).

Node.js

Node.js est une plateforme qui permet d'exécuter JavaScript côté serveur, idéale pour les applications web scalables et en temps réel.

PHP

PHP est un langage de script utilisé principalement pour le développement web côté serveur.

C

C est un langage de programmation bas niveau utilisé pour le développement de systèmes d'exploitation, d'applications et de logiciels embarqués.

C++

C++ est une extension du langage C, permettant la programmation orientée objet, et est utilisé dans les jeux vidéo, les logiciels systèmes et plus encore.

SQL

SQL est un langage de programmation utilisé pour interagir avec des bases de données relationnelles, permettant d'effectuer des requêtes, des mises à jour et plus encore.
